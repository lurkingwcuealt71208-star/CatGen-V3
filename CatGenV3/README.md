## System Requirements

- **Operating System:** Windows (Recommended)  
- **Python:** Version 3.14 or higher (Ensure **“Add Python to PATH”** is checked during installation)  
- **Zero-Setup:** The game automatically configures itself on first run  

> Currently, the game is only supported on desktop computers.  
> Mobile support will be added in future updates.  

**Python Resources:**  
- [Official Python Downloads](https://www.python.org/downloads)

---

## How to Run (Auto-Configuration & Self-Installing)

1. **Install Python:** Ensure you have Python 3.14+ installed. Make sure **“Add Python to PATH”** is checked.  

2. **Run the Game:**  
   - On Windows: Double-click **`run.bat`**  
   - The game will automatically:  
     1. Create a `.venv` (virtual environment) to keep your system clean  
     2. Install all required dependencies (e.g., `pygame-ce`)  
     3. Verify asset integrity and check for updates from GitHub  
     4. Launch the game immediately—no manual setup or command line required

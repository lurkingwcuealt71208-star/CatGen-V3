### How to Run (AutoConfig & Self-Installing)

1.  **Install Python:** Ensure you have Python 3.8+ installed (https://www.python.org/downloads). Make sure to check **"Add Python to PATH"** during installation.
2.  **Run the Game:**
    - On Windows: Double-click **`run.bat`**.
    - The game will automatically:
        a. Create a `.venv` (virtual environment) to keep your system clean.
        b. Install all required dependencies (like `pygame-ce`).
        c. Verify asset integrity and check for updates from GitHub.
        d. Launch the game immediately. No manual setup or command line required!

### Keybinds
- **Movement:** WASD (Shift to Sprint)
- **Meow:** 1
- **Inventory:** I
- **Chat:** T
- **Scent / Track / Bury:** F / E / B
- **Menu:** ESC
- **Fullscreen:** F11
- **Screenshot:** F12 (Saves to a "screenshots" folder on your Desktop)
- **Debug Info:** F3
